v2rayN 新手保姆级教程：从下载安装到节点配置全流程

v2rayN 是目前功能最强大、最受欢迎的图形化代理客户端之一。随着 7.x 版本的发布，它不仅在 Windows 上表现卓越，更通过 Avalonia 框架完美支持了 macOS 和 Linux 系统。它支持多种协议，包括 VMess、VLESS (Reality)、Shadowsocks、Trojan 及 Hysteria2。

如果你是第一次接触 v2rayN，本教程将手把手带你完成从下载到成功开启代理的全过程。

第一步：下载与安装

1. 获取软件

为了确保安全性与下载速度，建议通过以下渠道获取：

  • GitHub 官方发布页：2dust/v2rayN/releases

2. 下载哪个文件？

对于大多数 Windows 10/11 用户，我们推荐下载： v2rayN-windows-64-SelfContained.zip

这个包的优势在于：

  • 解压即用：内置了所有必需的核心组件。
  • 环境免安装：无需手动安装 Microsoft .NET 运行环境。

3. 安装与运行

  1. 将下载的压缩包解压到一个固定的位置（建议不要放在桌面或 C 盘，推荐放在 D 盘或 E 盘文件夹中）。
  2. 在解压后的文件夹中，找到 v2rayN.exe (Windows 用户) 或对应的运行图标。
  3. 重要：Windows 用户建议右键点击程序，选择 “以管理员身份运行”，以确保代理权限正常。

注意：如果运行报错提示缺少组件，请根据提示安装 .NET 8.0 Desktop Runtime。

第二步：添加节点订阅

v2rayN 本身只是一个工具，你需要导入有效的服务器节点（订阅链接）才能访问网络。

1. 获取订阅地址

通常你可以从自己购买的加速服务（机场）后台复制 V2Ray 订阅链接。

我们以扬帆云机场为例，购买订阅服务后登陆机场服务商的后台，点击订阅链接，复制订阅链接，如下图：

2. 导入软件

  1. 打开 v2rayN 主界面。
  2. 点击顶部菜单栏的 订阅分组，选择 订阅分组设置。
  3. 在弹出的窗口中点击 添加。
  4. 在 别名 处随便填写，在 可选地址 (url) 处粘贴你刚才复制的订阅链接。
  5. 点击 确定 保存。
  6. 回到主界面，点击 订阅分组 -> **更新全部订阅 (不通过代理)**。

操作完成后，你应该能看到列表中出现了一排节点。

第三步：开启代理与模式选择

这是最关键的一步，决定了电脑是否已经处于代理加速状态。

1. 选择节点

在节点列表中，左键点击选中一个想使用的节点，然后按键盘上的 **Enter (回车)**。当节点的别名变色或显示“活动”状态时，表示已选中。

2. 设置系统代理

在软件界面的最底部图标栏，找到以下三项关键设置：

  • 系统代理：将其设置为 自动配置系统代理。此时底部图标会变为红色。
  • 路由：将其设置为 绕过大陆。这可以确保访问百度、淘宝等国内网站时不走代理，访问 YouTube、Google 时才加速。

3. 测试网络

打开浏览器，尝试访问 Google。如果能正常打开，恭喜，配置已成功。

高级设置说明

TUN 模式

在软件底部可以找到 TUN 模式开关。

  • 作用：接管整机流量。对于一些不遵循系统代理的浏览器插件、游戏或特定软件非常有用。
  • 建议：普通网页浏览不需要开启，仅在某些软件无法正常代理时开启。

核心选择

v2rayN 支持切换 Xray-core、sing-box 等核心。目前大部分订阅链接都支持 Xray，保持默认即可。

常见问题排查 FAQ

1. 软件开启了但无法上网？

  • 检查系统时间是否准确（误差不能超过 90 秒，否则无法连接）。
  • 确认系统代理是否已设为“自动配置系统代理”。
  • 尝试更换其他节点进行连接。

2. 点击更新订阅失败？

  • 请确认你的网络能够访问订阅地址，或者尝试更换不同的网络环境。
  • 确认订阅链接是否过期或被服务商重置。
